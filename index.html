1. Computer e dispositivi

1.1 ICT

Tecnologie dell’Informazione e della Comunicazione. Permettono di archiviare, elaborare e trasmettere informazioni.

Esempi: e-commerce, e-banking, e-government, e-learning, smart working, tecnologie mobili.

1.2 Hardware

  • Tipi di computer: Desktop (fisso), Laptop/Notebook (portatile), Tablet
  • Dispositivi: Smartphone, lettori multimediali, fotocamere digitali
  • Processore (CPU): Esegue i calcoli. Velocità misurata in GHz
  • RAM: Memoria temporanea (volatile) – più RAM = computer più veloce
  • Memoria di massa: Disco fisso, USB, schede di memoria, CD/DVD (permanente)
  • Periferiche: Stampanti (laser, inkjet), schermi (LCD/LED), scanner, tastiera, mouse/trackpad, webcam, altoparlanti, microfono
  • Porte comuni: USB, HDMI, Ethernet (rete), FireWire

2. Desktop, icone, impostazioni

Il desktop è la scrivania digitale. La barra delle applicazioni permette di passare tra programmi.

  • Gestire icone: selezionare, spostare, creare/rinominare collegamenti
  • Gestione finestre: aprire, minimizzare, massimizzare, ridimensionare, chiudere
  • Impostazioni: data e ora, volume, sfondo, risoluzione schermo, lingue tastiera
  • Installare/disinstallare applicazioni
  • Collegare e scollegare dispositivi USB in modo sicuro
  • Catturare schermo (Print Screen)

3. Testi e stampe

Uso di programmi di elaborazione testi (es. Word):

  • Aprire, scrivere, copiare/incollare testo e immagini
  • Salvare documenti con nome appropriato

Gestione stampanti:

  • Installare stampante
  • Impostare stampante predefinita
  • Stampare documenti e gestire la coda di stampa

4. Gestione di file

Il sistema operativo organizza i dati in unità, cartelle e file (struttura gerarchica).

  • Creare, rinominare, copiare, spostare, eliminare file e cartelle
  • Visualizzare proprietà (nome, dimensione, data)
  • Cercare file per nome o contenuto
  • Tipi di file: .txt, .docx, .xlsx, .pptx, .pdf, .jpg, .mp3, .mp4, .zip
  • Comprimere ed estrarre file (ZIP)
  • Supporti di memoria: disco fisso, USB, cloud, CD/DVD

5. Reti

Rete: insieme di computer collegati per condividere dati e dispositivi.

  • Internet: rete globale (WWW, email, VoIP...)
  • Tipi di connessione: Wi-Fi, cavo, mobile, satellite
  • ISP (Internet Service Provider)
  • Velocità: bps, Kbps, Mbps, Gbps (download e upload)
  • Reti wireless: protette (con password) o aperte

6. Sicurezza e benessere

  • Password sicure: lunghe, con lettere, numeri, simboli. Cambiarle regolarmente
  • Firewall: protegge da accessi non autorizzati
  • Backup: copie di sicurezza dei dati
  • Malware: virus, worm, trojan, spyware. Usare antivirus
  • Salute: posture corretta, pause regolari, illuminazione adeguata
  • Informatica verde: spegnimento automatico, riciclo dispositivi
  • Opzioni di accessibilità: zoom, screen reader, contrasto elevato
