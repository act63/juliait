1. Computer e dispositivi

1.1 ICT

Tecnologie dell’Informazione e della Comunicazione. Servono per archiviare, elaborare e trasmettere informazioni. Esempi: e-commerce, e-banking, e-government, e-learning, smart working.

1.2 Hardware

  • Tipi di computer: desktop, laptop (portatile), tablet
  • Dispositivi: smartphone, lettori multimediali, fotocamere digitali
  • Processore (CPU): esegue i calcoli, velocità in GHz
  • RAM: memoria temporanea (volatile) – più RAM = prestazioni migliori
  • Memoria di massa: disco fisso, USB, schede di memoria, CD/DVD (permanente)
  • Periferiche: stampanti, schermi, scanner, tastiera, mouse, webcam, altoparlanti, microfono
  • Porte: USB, HDMI, Ethernet, FireWire

2. Desktop, icone, impostazioni

Il desktop è l'area di lavoro principale. La barra delle applicazioni permette di lanciare programmi rapidamente.

  • Gestire icone e collegamenti
  • Usare e ridimensionare finestre
  • Modificare data/ora, volume, sfondo, risoluzione
  • Installare/disinstallare applicazioni
  • Collegare dispositivi USB in sicurezza

Flashcards - Modulo 2

3. Testi e stampe

Usare applicazioni di elaborazione testi (aprire, scrivere, copiare, incollare, salvare).

Gestire stampanti: installare, impostare come predefinita, stampare documenti, gestire la coda di stampa.

Flashcards - Modulo 3

4. Gestione di file

Struttura gerarchica: unità → cartelle → file.

  • Creare, rinominare, copiare, spostare, eliminare file e cartelle
  • Cercare file per nome o contenuto
  • Comprimere ed estrarre file (.zip)
  • Tipi di supporto: disco fisso, USB, cloud

Flashcards - Modulo 4

5. Reti

Una rete permette di condividere dati e dispositivi.

  • Internet, intranet, VPN
  • Connessioni: Wi-Fi, cavo, mobile
  • ISP (fornitore di connessione)
  • Velocità: download e upload (Mbps, Gbps)

Flashcards - Modulo 5

6. Sicurezza e benessere

  • Password forti, firewall, aggiornamenti regolari, backup
  • Malware: virus, worm, trojan, spyware
  • Salute: pause, postura corretta, illuminazione
  • Informatica verde: risparmio energetico, riciclo
  • Accessibilità: zoom, screen reader, contrasto elevato
