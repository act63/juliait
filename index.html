1. Computer e dispositivi

1.1 ICT

Tecnologie dell’Informazione e della Comunicazione. Servono per archiviare, elaborare e trasmettere informazioni. Esempi principali: e-commerce, e-banking, e-government, e-learning, smart working, applicazioni di produttività.

1.2 Hardware

  • Tipi di computer: desktop, laptop (portatile), tablet
  • Dispositivi: smartphone, lettori multimediali, fotocamere digitali
  • Processore (CPU): esegue calcoli, velocità misurata in GHz
  • RAM: memoria temporanea (si perde quando spegni)
  • Memoria di massa: disco fisso, USB, schede memoria, CD/DVD (dati permanenti)
  • Periferiche principali: stampanti, schermi, scanner, tastiera, mouse, webcam, altoparlanti, microfono
  • Porte comuni: USB, HDMI, Ethernet

1.3 Software e licenze

Sistemi operativi (Windows, macOS, Linux, Android, iOS) e applicazioni. Licenze: proprietaria, open source, freeware, shareware, EULA (contratto utente).

2. Desktop, icone, impostazioni

Il desktop è l'area di lavoro principale. La barra delle applicazioni permette di passare tra i programmi aperti.

  • Gestire icone: selezionare, spostare, creare collegamenti
  • Gestione finestre: minimizzare, massimizzare, ridimensionare, chiudere
  • Impostazioni: data/ora, volume, sfondo, risoluzione schermo, lingue tastiera
  • Installare/disinstallare applicazioni
  • Collegare e scollegare dispositivi USB correttamente

Flashcards - Modulo 2

3. Testi e stampe

Usare programmi di elaborazione testi: aprire, inserire testo, copiare/incollare, salvare documenti.

Gestione stampanti: installare, impostare come predefinita, stampare documenti, gestire la coda di stampa.

Flashcards - Modulo 3

4. Gestione di file

Il sistema organizza i dati in unità, cartelle e file (struttura gerarchica).

  • Creare, rinominare, copiare, spostare, eliminare file/cartelle
  • Visualizzare proprietà, cambiare vista (icone, lista, dettagli)
  • Ricercare file per nome o contenuto
  • Comprimere ed estrarre file (.zip)
  • Tipi di supporti: disco fisso, USB, cloud, CD/DVD

Flashcards - Modulo 4

5. Reti

Una rete permette di condividere dati e dispositivi in sicurezza.

  • Internet, intranet, VPN
  • Connessioni: Wi-Fi, cavo, mobile, satellite
  • ISP (Internet Service Provider)
  • Velocità: download e upload (Mbps, Gbps)

Flashcards - Modulo 5

6. Sicurezza e benessere

  • Password sicure (lunghe, con numeri e simboli), firewall, aggiornamenti, backup
  • Malware: virus, worm, trojan, spyware – usare antivirus
  • Salute: pause regolari, postura corretta, illuminazione adeguata
  • Informatica verde: risparmio energetico, riciclo dispositivi
  • Accessibilità: zoom, screen reader, contrasto elevato
