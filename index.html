ICDL Computer Essentials

Tutto il materiale principale del manuale organizzato per modulo.

1. Computer e dispositivi

1.1 ICT

Tecnologie dell’Informazione e della Comunicazione. Esempi: e-commerce, e-banking, e-government, e-learning, smart working.

1.2 Hardware

  • Tipi di computer: Desktop, Laptop, Tablet
  • Processore (CPU), RAM, Memoria di massa
  • Periferiche: tastiera, mouse, stampante, schermo, scanner, webcam, altoparlanti, microfono
  • Porte comuni: USB, HDMI, Ethernet

1.3 Software e Licenze

Sistemi operativi (Windows, Android, ecc.) e applicazioni. Tipi di licenze: proprietaria, open source, freeware, shareware.

2. Desktop, icone, impostazioni

Gestione del desktop, icone, finestre, impostazioni (data, volume, risoluzione, lingue), installazione programmi, collegamento dispositivi USB.

3. Testi e stampe

Uso di elaboratori di testi, copia/incolla, salvataggio documenti, installazione e uso della stampante, gestione coda di stampa.

4. Gestione di file

Organizzazione di file e cartelle, copia/spostamento/elimina, ricerca file, compressione, tipi di supporti di memoria.

5. Reti

Concetti di rete, Internet, Wi-Fi, provider, velocità di trasferimento, reti protette.

6. Sicurezza e benessere

Password sicure, firewall, backup, malware, tutela della salute, informatica verde, accessibilità.
