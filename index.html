1. Computer e dispositivi

1.1 ICT

Information and Communication Technology – Tecnologie dell’Informazione e della Comunicazione.

Esempi: e-commerce, e-banking, e-government, e-learning, smart working.

1.2 Hardware

  • Tipi di computer: desktop, laptop (portatile), tablet
  • Dispositivi: smartphone, lettori multimediali, fotocamere digitali
  • Componenti principali: Processore (CPU), RAM (memoria volatile), Memoria di massa (disco fisso, USB, ecc.)
  • Periferiche: stampanti, schermi, scanner, tastiera, mouse, webcam, altoparlanti, microfono
  • Porte comuni: USB, HDMI, Ethernet, FireWire

2. Desktop, icone, impostazioni

Scopo del desktop e della barra delle applicazioni.

  • Gestione icone (selezionare, spostare, creare collegamenti)
  • Uso delle finestre (ridurre, massimizzare, chiudere, passare da una all’altra)
  • Impostazioni: data/ora, volume, sfondo, risoluzione, lingue tastiera
  • Installare/disinstallare applicazioni, collegare dispositivi USB

3. Testi e stampe

Uso di applicazioni di elaborazione testi (aprire, inserire testo, copiare/incollare, salvare).

Gestione stampanti: installare, impostare predefinita, stampare documenti, gestire coda di stampa.

4. Gestione di file

Struttura gerarchica di file e cartelle.

  • Creare, rinominare, copiare, spostare, eliminare file/cartelle
  • Tipi di file comuni (testo, foglio elettronico, PDF, immagine, ecc.)
  • Ricerca file, visualizzazione (icone, lista, dettagli)
  • Compressione ed estrazione di file

5. Reti

Concetti di rete, Internet, intranet, VPN.

  • Connessioni: Wi-Fi, cavo, mobile, satellite
  • Provider (ISP), velocità (bps, Mbps, Gbps)
  • Scaricamento e caricamento

6. Sicurezza e benessere

  • Password sicure, firewall, backup
  • Malware (virus, worm, trojan, spyware)
  • Informatica verde, posture corretta, pause regolari
  • Accessibilità (zoom, screen reader, contrasto elevato)
