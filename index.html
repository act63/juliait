ICDL Computer Essentials

Tutto il materiale principale del manuale è qui, organizzato per modulo.

1. Computer e dispositivi

1.1 ICT

Tecnologie dell’Informazione e della Comunicazione. Permettono di archiviare, elaborare e trasmettere informazioni.

Servizi principali: e-commerce, e-banking, e-government, e-learning, smart working, tecnologie mobili, applicazioni di produttività.

1.2 Hardware

  • Tipi di computer: desktop, laptop (portatile), tablet
  • Dispositivi: smartphone, lettori multimediali, fotocamere digitali
  • Processore (CPU): esegue i calcoli, velocità in GHz
  • RAM: memoria temporanea (volatile) – influisce sulle prestazioni
  • Memoria di massa: disco fisso, USB, schede memoria, CD/DVD, cloud
  • Periferiche: stampanti (laser, inkjet), schermi (LCD/LED, touchscreen), scanner, tastiera (QWERTY), mouse, trackpad, webcam, altoparlanti, microfono, docking station
  • Porte: USB, HDMI, Ethernet, FireWire, seriale, parallela

1.3 Software e licenze

  • Software: sistemi operativi (Windows, macOS, Linux, Android, iOS) e applicazioni
  • Tipi di licenze: proprietaria, open source, freeware, shareware, trial
  • EULA: contratto con l’utente finale

1.4 Avvio e spegnimento

Accensione con account Microsoft, login con password, spegnimento, riavvio, disconnessione.

2. Desktop, icone, impostazioni

  • Scopo del desktop e della barra delle applicazioni
  • Gestione icone: selezionare, spostare, creare collegamenti
  • Gestione finestre: aprire, minimizzare, massimizzare, ridimensionare, chiudere, passare da una all’altra
  • Impostazioni: data/ora, volume, sfondo, risoluzione, lingue tastiera
  • Chiudere programmi bloccati, installare/disinstallare app
  • Collegare dispositivi USB in sicurezza
  • Catturare schermo

3. Testi e stampe

  • Usare applicazioni di elaborazione testi: aprire, inserire testo, copiare/spostare, incollare immagine schermo, salvare
  • Gestione stampanti: installare, impostare predefinita, stampare, gestire coda di stampa

4. Gestione di file

  • Struttura gerarchica: unità → cartelle → file
  • Creare, rinominare, copiare, spostare, eliminare file/cartelle
  • Visualizzare proprietà, cambiare visualizzazione (icone, lista, dettagli)
  • Tipi di file comuni (.txt, .docx, .xlsx, .pdf, .jpg, .zip, ecc.)
  • Ricerca file per nome, contenuto, data
  • Comprimere ed estrarre file
  • Supporti di memoria: KB, MB, GB, TB

5. Reti

  • Concetto di rete: condividere dati e dispositivi in sicurezza
  • Internet, intranet, VPN
  • Connessioni: linea telefonica, mobile, cavo, Wi-Fi, satellite
  • ISP (provider), velocità di download/upload
  • Reti wireless protette vs aperte

6. Sicurezza e benessere

  • Password forti, firewall, aggiornamenti software, backup
  • Malware: virus, worm, trojan, spyware – usare antivirus
  • Salute: pause regolari, postura corretta, illuminazione giusta
  • Informatica verde: risparmio energetico, riciclo
  • Accessibilità: zoom, screen reader, contrasto elevato
